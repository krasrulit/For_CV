# Econometrics Projects

This folder contains applied econometrics assignments prepared as portfolio examples.

## Urban Wage Premium

Path: `urban_wage_premium/`

This project studies wage differences associated with urban labor markets. The analysis is provided as a Jupyter notebook, with the final report and task statement included as PDFs.

Files:

- `analysis.ipynb` - empirical analysis notebook
- `report.pdf` - final written report
- `task.pdf` - assignment statement
- `wage_gap.xlsx` - supporting dataset
- `NLS_Investigator_Search.pdf` - supporting source material

## Platform Price Parity

Path: `platform_price_parity/`

This project analyzes platform price parity using applied econometric methods. The folder includes the analysis notebook, final report, task statement, and supporting Stata datasets.

Files:

- `analysis.ipynb` - empirical analysis notebook
- `report.pdf` - final written report
- `task.pdf` - assignment statement
- `data_mac_sr.dta` - supporting data
- `data_mac_synth.dta` - supporting data
- `data_mr_placebo.dta` - supporting data
