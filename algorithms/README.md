# Algorithms

This folder collects my solutions to algorithm problems.
It is split into two parts:
1. **Problems from Yandex's algorithms training (Training 1, HW 1)**
2. **LeetCode "sum" problems**

Each problem is laid out as: *problem statement* + *my solution*.

---

## Contents

### [Yandex_algorithms/training_1](Yandex_algorithms/training_1)
- Homework 1 from Yandex's **"Algorithms Training"** course.
- Covers classic problems on I/O, strings, arrays, basic data structures, and algorithmic techniques.
- Format: for each problem, a file with the statement and my solution.

This part shows my ability to solve basic algorithmic problems in a competitive-programming format.

---

### [sum_algorithms](sum_algorithms)
- A set of **LeetCode** problems on different "sum" variations:
  - Two Sum
  - 3Sum
  - 4Sum, and others
- For each problem: a file with the statement and a file with my solution.
- The solutions cover:
  - brute-force and optimal approaches,
  - use of dictionaries,
  - the two-pointer technique,
  - sorting and time/memory optimization.

This part shows knowledge of **algorithmic patterns** for working with sums, arrays, and hash tables.
